# theGoodTube webapp

## How to use it

First of all you need to have a working theGoodTube API instance.
Then, you need to setup a web server (like Apache or Nginx) to serve a production build version of this webapp

## Project setup (for development)

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Run your tests

```
yarn run test
```

### Lints and fixes files

```
yarn run lint
```
